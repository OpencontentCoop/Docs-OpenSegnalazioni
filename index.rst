Manuale OpenSegnalazioni
########################

\ |LINK1|\  è uno strumento web che consente alla Pubblica Amministrazione di \ |STYLE0|\ . 

I \ |STYLE1|\  possono inserire nel sistema le loro \ |STYLE2|\ , che vengono prese in carico dal personale dell’ente in base alle competenze. La gestione della segnalazione è \ |STYLE3|\  per il cittadino che, in ogni momento, può verificarne lo stato.

Le \ |STYLE4|\  possono avere un confronto diretto con i cittadini, ottimizzare i flussi interni per la gestione della pratiche ma anche monitorare l’andamento delle performance di risoluzione delle segnalazioni. 

L’applicazione può inoltre generare \ |STYLE5|\  per misurare le performance relative alla gestione delle segnalazioni dei cittadini.
